## Customer & Sales Performance Analysis

## Overview

This project analyzes customer behavior and sales performance to uncover customer purchasing patterns and revenue trends using SQL, Python, and Microsoft Power BI.

The objective was to identify sales performance, customer behavior, most performing category, age group of top customers, and gender of most customers to create an interactive dashboard for decision making.

## Objective

- Analyze overall sales performance
- Identify the best-performance product categories
- Determine customer purchasing pattern
- Track monthly sales trends
- Visualize key business metrics
- Present findings through an interactive dashboard

## Tools Used

- Python
    - Pandas
    - Matplotlib
- Microsoft SQL Server(SSMS)
- Microsoft Power BI
- Microsoft Excel

## Project workflow

1. Cleaned and prepared the data using Python (Pandas).
2. Imported the dataset into SQL Server.
3. Performed SQL queries to answer business questions.
4. Create visualizations with matplotlib.
5. Built an interactive Power BI dashboard
6. Wrote a business report summarizing the findings.

## SQL Analysis

The SQL queries include:

- Total Revenue
- Average Purchasing Amount
- Total Number of Transactions
- Revenue by Product Category
- Revenue by Gender
- Monthly Revenue trend
- Revenue by Age Group
- Highest Spending Product Category
- Average Customer Age

## Power BI Dashboard

The dashboard includes: 

- Total Revenue KPI
- Average Purchase KPI
- Total Transaction KPI
- Average Customer Age KPI
- Monthly Revenue Chart
- Revenue by Region Chart
- Revenue by Gender Chart
- Top Customers Chart
- Age Distribution Chart
- Gender Slicer
- Product Category Slicer
- Month Slicer

## Key Insights

- Electronics generated highest Revenue
- Revenue peaked in May and October
- Customers aged 18-25 contributed to the highest revenue
- Male and female spent similar amounts
